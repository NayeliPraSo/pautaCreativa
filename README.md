# Pauta Creativa

Sitio web oficial de **Pauta Creativa**, desarrollado con **Astro**.

Este proyecto tiene como objetivo renovar la presencia digital de la agencia mediante un sitio web moderno, rápido y optimizado para ofrecer una mejor experiencia de usuario, alto rendimiento y una estructura escalable que facilite su mantenimiento y evolución.

## Características

- ⚡ Desarrollado con Astro
- 📝 TypeScript
- 🚀 Optimizado para rendimiento y SEO
- 📱 Diseño responsive
- 🧩 Arquitectura basada en componentes
- 🎨 Estilos organizados y escalables
- 📈 Preparado para futuras integraciones

## Requisitos

- Node.js 22 o superior
- npm

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/NayeliPraSo/pautaCreativa
```

Instala las dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm run dev
```

Abre el navegador en:

```
http://localhost:4321
```

## Scripts disponibles

| Comando           | Descripción                           |
| ----------------- | ------------------------------------- |
| `npm run dev`     | Inicia el servidor de desarrollo.     |
| `npm run build`   | Genera la versión de producción.      |
| `npm run preview` | Previsualiza la aplicación compilada. |
